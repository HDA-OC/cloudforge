# Tools

Guides for tools and technologies used in CloudForge.

## Core Technologies

- **Next.js 14** — React framework with App Router
- **TypeScript** — Type-safe JavaScript
- **Stripe** — Payment and identity verification
- **Twitter API** — OAuth 2.0 authentication

## Development Tools

- **Docker** — Containerized development (REQUIRED)
- **pnpm/npm** — Package management
- **ESLint** — Code linting
- **Prettier** — Code formatting
