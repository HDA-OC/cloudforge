# CloudForge Documentation

User-facing documentation for CloudForge.

## What is CloudForge?

CloudForge provides identity verification for Twitter users through Stripe Identity.

## How It Works

1. **Authenticate** — Sign in with your Twitter account
2. **Verify** — Complete identity verification via Stripe
3. **Done** — Your Twitter account is now linked to a verified identity
